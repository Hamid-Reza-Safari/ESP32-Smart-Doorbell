# ESP32 Smart Door Bell

A simple WiFi-based smart doorbell using ESP32 and a desktop notification client.

## Features

- Real-time WebSocket notifications
- Custom sound alerts
- Auto reconnect
- System tray support
- Windows desktop client

## Hardware

- ESP32 WROOM-32
- Touch sensor module
- Passive buzzer

## Wiring

| Module | ESP32 |
|----------|----------|
| Touch SIG | GPIO4 |
| Touch VCC | 3.3V |
| Touch GND | GND |
| Buzzer + | GPIO18 |
| Buzzer - | GND |
